PayflowNETAPISubmitTransaction Method

Submits a transaction to Payflow Server.

PayflowNETAPI is used to submit a Name-value pair or XMLPay request to PayPal payment gateway for online payment processing. The response returned is the string value of the response from the PayPal payment gateway.
