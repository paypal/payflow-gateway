Remarks
The Void transaction prevents a transaction from being settled, but does not release the authorization (hold on funds) on the cardholder’s account. Delayed Capture, Sale, Credit, Authorization, and Voice Authorization transactions can be voided. A Void transaction cannot be voided. The Void must occur prior to settlement.
