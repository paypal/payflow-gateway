PayPal.Payments.Transactions Namespace

NameSpace doc for Transactions namespace
Classes
  ClassDescription
AuthorizationTransaction
This class is used to create and perform an Authorization Transaction.
BaseTransaction
This class is the base class for all transaction objects. It has methods for generating the transaction request, sending it to the server and obtaining the response. For an usage of this class, please see the examples in SamplesCS + SamplesVB folders namely DOSale_Base.cs and DOSale_Base.vb.
BuyerAuthTransaction
This abstract class serves as base class for Buyer auth transactions.
BuyerAuthVATransaction
This class is used to create and perform a Validate Authentication transaction. Validate Authentication is the second step of Buyer authentication process.
BuyerAuthVETransaction
This class is used to create and perform a Verify Enrollment transaction. Verify Enrollment is the first step of Buyer authentication process.
CaptureTransaction
This class is used to perform a capture transaction.
CreditTransaction
This class is used to create and perform an Credit Transaction.
FraudReviewTransaction
This class is used to perform a fraud review transaction.
InquiryTransaction
This class is used to perform an inquiry transaction.
OrderTransaction
This class is used to create and perform an Order Transaction for Express Checkout. An Order transaction represents an agreement to pay one or more authorized amounts up to the specified total over a maximum of 29 days.
RecurringAddTransaction
This class is used to perform a recurring transaction with add action.
RecurringCancelTransaction
This class is used to perform a recurring transaction with cancel action.
RecurringInquiryTransaction
This class is used to perform a recurring transaction with inquiry action.
RecurringModifyTransaction
This class is used to perform a recurring transaction with modify action.
RecurringPaymentTransaction
This class is used to perform a recurring transaction with payment action.
RecurringReActivateTransaction
This class is used to perform a recurring transaction with reactivate action.
RecurringTransaction
This is the base class of all different recurring action transactions.
ReferenceTransaction
This class is used as base class for all reference transactions.
SaleTransaction
This class is used to create and perform a Sale Transaction.
VoiceAuthTransaction
This class is used to perform a voice authorization transaction.
VoidTransaction
This class is used to perform a void transaction.
