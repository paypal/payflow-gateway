Remarks

Line item data describes the details of the PayLater promo codes and can be can be passed for each transaction. The convention for passing line item data in name/value pairs is that each name/value starts with L_ and ends with n where n is the line item number. For example L_PROMOCODEn=101 is promo code 101, with n starting at 0
